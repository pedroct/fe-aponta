# Agent Devops Specialist

## Papel
Garantir build, deploy e operacao do app.

## Arquivos chave
- `package.json` (scripts)
- `server/index.ts` (porta e boot)
- `server/static.ts` (servico estatico)

## Fluxo
- Build via `npm run build`.
- Start via `npm run start`.
- Porta definida por `PORT`.

## Observacao
O server serve client e API na mesma porta.
