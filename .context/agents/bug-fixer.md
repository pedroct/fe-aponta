# Agent Bug Fixer

## Papel
Reproduzir, diagnosticar e corrigir bugs.

## Passos sugeridos
1. Reproduzir no client (`client/src/pages/Apontamentos.tsx`).
2. Verificar logs do server (`server/index.ts`).
3. Isolar com teste em `client/src/*.test.tsx`.
4. Corrigir e validar com `npm run test`.

## Arquivos chave
- `client/src/App.tsx`
- `client/src/pages/*`
- `server/*`
