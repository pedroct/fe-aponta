# Agent Feature Developer

## Papel
Implementar novas funcionalidades no client e API.

## Passos sugeridos
1. Definir UI em `client/src/pages`.
2. Criar ou ajustar chamadas em `client/src/lib/queryClient.ts`.
3. Implementar rotas em `server/routes.ts`.
4. Adicionar testes em `client/src/*.test.tsx`.

## Convencoes
- Usar componentes de `client/src/components/ui`.
- Reutilizar hooks existentes.
