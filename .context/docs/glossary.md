# Glossario

## Termos de negocio
- **Apontamento**: registro de horas trabalhadas.
- **Atividade**: tipo de trabalho associado ao apontamento.

## Termos tecnicos
- **React Query**: gerencia cache e chamadas HTTP no client.
- **Wouter**: roteador leve para React.
- **Toast**: feedback visual de acao (sucesso/erro).

## Artefatos do projeto
- **Storage em memoria**: implementacao temporaria em `server/storage.ts`.
- **Schema compartilhado**: tipos e tabelas em `shared/schema.ts`.
