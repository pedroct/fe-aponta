# Ferramentas

## Build e runtime
- Vite (client)
- Express (server)
- TSX para executar TypeScript no server

## Styling
- Tailwind CSS e utilitarios em `client/src/index.css`

## Banco e schema
- Drizzle ORM (apenas schema compartilhado)

## Testes
- Vitest + Testing Library

## Scripts relevantes
- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run check`
